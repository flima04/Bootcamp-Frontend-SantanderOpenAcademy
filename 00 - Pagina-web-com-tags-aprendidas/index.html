A Linguagem que Estrutura a Web: Conheça o HTML

Navegar na internet, ler notícias, ver vídeos e interagir em redes sociais são ações rotineiras que dependem de uma tecnologia fundamental: o HTML. Essencial para a construção de qualquer página web, essa linguagem de marcação é o alicerce sobre o qual todo o conteúdo da internet é construído.

O Surgimento do HTML

voltar

A história do HTML começa no final da década de 1980 e início de 1990, em um dos mais importantes centros de pesquisa do mundo, a Organização Europeia para a Pesquisa Nuclear (CERN). O físico britânico Tim Berners-Lee, enquanto trabalhava no CERN, sentiu a necessidade de criar um sistema que permitisse o compartilhamento e a fácil visualização de documentos de pesquisa entre cientistas de diferentes instituições.

"A ideia original da web era que ela fosse um espaço colaborativo onde você pudesse se comunicar através do compartilhamento de informações." Tim Berners-Lee

Em 1991, Berners-Lee propôs um sistema baseado em hipertexto, que permitia a ligação entre diferentes documentos. Para isso, ele criou o HTML (HyperText Markup Language), uma linguagem simples de marcação que definia a estrutura de um documento, como títulos, parágrafos e links. Juntamente com o protocolo HTTP(Hypertext Transfer Protocol) e o conceito de World Wide Web, o HTML de Berners-Lee revolucionou a forma como a informação era partilhada, dando origem à internet como a conhecemos hoje.

Para que Serve o HTML?

voltar

Ao contrário do que muitos pensam, o HTML não é uma linguagem de programação, mas sim uma linguagem de marcação. Sua principal função é estruturar o conteúdo de uma página da web. Pense no HTML como a espinha dorçal o "esqueleto" de um site. Ele utiliza um sistema de "tags" (etiquetas) para indicar ao navegador como exibir os elementos na tela.

É com o HTML que se define o que é um título, um parágrafo, uma lista, onde uma imagem deve ser inserida ou para qual outra página um link deve apontar. Ele forma a camada fundamental de uma página, sobre a qual outras tecnologias, como o CSS (para estilização e aparência) e o JavaScript (para interatividade), atuam para criar a experiência completa do usuário.

Principais Tags do HTML

voltar

As tags são os comandos básicos do HTML, escritos entre os sinais de menor que (<) e maior que (>). Embora existam centenas de tags, algumas são essenciais e utilizadas em praticamente todas as páginas da web.

Tags Estruturais:

  • <html>: Define o início e o fim de um documento HTML.
  • <head>: Contém informações sobre o documento que não são exibidas diretamente no conteúdo da página, como o título, metadados e links para folhas de estilo.
  • <body>: Engloba todo o conteúdo visível da página, como textos, imagens e vídeos.

Tags de Texto e Conteúdo:

  • <h1> a <h6>: Definem títulos e subtítulos, sendo <h1> o de maior importância.
  • <p>: Representa um parágrafo.
  • <a>: Cria um hiperlink para outra página ou recurso.
  • <img>: Insere uma imagem na página.
  • <ul>, <ol> e <li>: Usadas para criar listas não ordenadas (com marcadores) e ordenadas (com números), respectivamente. Cada item da lista é definido pela tag <li>.

Tags Semânticas (Introduzidas no HTML5):

  • <header>: Define o cabeçalho de uma página ou de uma seção.
  • <footer>: Representa o rodapé da página.
  • <article>: Delimita um conteúdo independente e completo, como um post de blog.
  • <section>: Agrupa conteúdos relacionados tematicamente.
